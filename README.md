# Blinkit Sales Analytics & Forecasting Dashboard

**Customized and Enhanced** sales analytics dashboard for **Blinkit** - India's leading quick-commerce platform. While many Blinkit dashboards exist publicly, this project stands out with **Python-powered sales forecasting** integrated into a professional Power BI dashboard.

## ✨ Key Highlights

- **Unique Addition**: Built and integrated **time-series sales forecasting** using Python, extending beyond standard dashboards
- **KPI Cards**: Total Sales ($1M), Avg Sales ($141), Total Items (9K), Avg Rating (4.0)
- **Interactive Data Table** with advanced filters
- **Historical Trend Analysis** (2012–2022)
- **Future Sales Forecast** (2023–2030) — Projected to reach ~$1.8M by 2028
- **Rich Visualizations**: Donut charts, bar charts, line trends, and tier-wise outlet analysis

## 🛠️ Tech Stack

- **Python** → Data cleaning, EDA, and **Sales Forecasting**  
  (pandas, numpy, Prophet/ARIMA/Statsmodels, matplotlib/seaborn)
- **Power BI** → Interactive dashboard, DAX calculations, and stunning visualizations
- End-to-End Integration: Python model output connected to Power BI

## 📊 Key Insights

- Strong dominance of **Soft Drinks** category
- Impact of outlet establishment year on sales performance
- Performance comparison across Supermarket Types, Grocery Stores & Tiers
- Actionable future demand predictions for inventory and expansion planning

## 🎯 Project Differentiation

This project is **not just another Blinkit dashboard** — the main value comes from:
- Developing a custom forecasting model in Python
- Seamlessly integrating forecast results into Power BI
- Creating a production-ready analytics solution
